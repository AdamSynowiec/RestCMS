REST API Documentation v1.0.0
Basic API information API Base URL https://restcms.ct8.pl
Installation
https://restcms.ct8.pl/setup?auth={$secret_key}
Initial Installation
  • Authorization by Key: admin_setup
Subsequent Instances
  • Authorization by a key that is generated within the auth table
Authorization
https://restcms.ct8.pl?auth={$secret_key}
Response: 
 {
    "status": 200,
    "message": "success"
 }
Table operations
Pobieranie listy utworzonych tabeli https://restcms.ct8.pl/api/get/table
Response: 
 {
    "status": 200,
    "message": "success"
 }
Tworzenie nowej tabeli https://restcms.ct8.pl/api/post/table
Body: 
 {
    "table":"tableName"
 }
Response: 
 {
    "status": 200,
    "message": "success"
 }
Deleting an existing table https://restcms.ct8.pl/api/delete/table
Body:

    "table":"tableName"
 }
Response:

 }
Attribute operations
Create a new attribute in a table https://restcms.ct8.pl/api/post/attribute
Object key Required Object value Description
type Yes
  • Number
  • String
  • Text
 Type of object represented
name Yes
  • String
 Name of the represented object
Data: 
 {
    "table": "tableName",
    "attribute": {
        "type": "number",
        "name": "name"
    }
 }
Response: 
 {
    "status": 200,
    "message": "success"
 }
Deleting an existing attribute from the table https://restcms.ct8.pl/api/delete/attribute/{$table}/delete?data={object_data}
